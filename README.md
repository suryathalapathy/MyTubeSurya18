# MyTube APK

A lightweight YouTube WebView app — fullscreen, no browser chrome.

## To block ads
Set your phone DNS to AdGuard: `94.140.14.14` / `94.140.15.15`
Settings → Wi-Fi → Modify → Advanced → DNS

## Build via GitHub Actions (Free)
1. Create a new GitHub repo
2. Upload all these files
3. Go to Actions tab → Run "Build MyTube APK"
4. Download the APK from the Artifacts section

## Build locally (Android Studio)
1. Open this folder in Android Studio
2. Click Build → Build Bundle(s)/APK(s) → Build APK(s)
3. APK will be at: app/build/outputs/apk/debug/app-debug.apk

## Ad Blocking (no DNS needed)
Install AdGuard for Android alongside this app — it blocks ads system-wide.
